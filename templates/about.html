Work in progress. This website is a personal project to make learning Finnish vocabulary clearer, better structured, and more enjoyable. It combines dictionary-style information with authentic usage data, including example sentences and common word combinations drawn from real Finnish subtitle material (such as films and TV series).

What the site offers

  • Word meanings with multiple translations (where available)
  • Topic categories and subcategories for organizing vocabulary
  • Level filtering to focus on easier or more advanced vocabulary
  • Interactive study views: table, card view, and flashcards
  • Example sentences with English translations (in progress)
  • Word relationships (e.g. related meanings, variants, shared roots) (in progress)

The level system

Every word is assigned a difficulty level from 1 to 5 (plus “+” for unranked or special items). These levels are not official CEFR levels — they are a personal heuristic intended to help structure learning:

  • Level 1: Very common, essential beginner vocabulary
  • Level 2: Basic everyday words
  • Level 3: Intermediate vocabulary
  • Level 4: Less common or more specific words
  • Level 5: Advanced, formal, or abstract vocabulary
  • Level +: Not ranked yet

You can filter pages by level, and the selection stays consistent as you navigate.

Data sources & attribution

Many basic definitions and translations originate from Wiktionary. Wiktionary content is licensed under CC BY-SA 4.0, so this project may reuse and adapt it with attribution. Other content (such as curation, organization, and some examples) is added as part of this project.

Notes

This project is evolving continuously. Some parts of the vocabulary set and example coverage are still limited, and the site may change as new features are added.
