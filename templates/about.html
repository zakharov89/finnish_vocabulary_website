This website is a personal project created to make learning Finnish vocabulary clearer, better structured, and more enjoyable. It aims to combine dictionary-quality information with interactive tools for studying words in meaningful ways.

What the site offers

  • Word meanings with multiple translations
  • Topic categories and subcategories that organize vocabulary
  • Level filtering to focus on easier or more advanced vocabulary
  • Interactive views: table, card view, and flashcards
  • Example sentences with English translations (coming soon)
  • Word and meaning relations such as synonyms or shared roots (coming soon)

The level system

Every word is assigned a difficulty level from 1 to 5 (plus “+” for unranked or special items). These levels are not official CEFR levels — they are a personal heuristic to help structure learning:

  • Level 1: Very common, essential beginner vocabulary
  • Level 2: Basic everyday words
  • Level 3: Intermediate vocabulary
  • Level 4: Less common or more specific words
  • Level 5: Advanced, formal, or abstract vocabulary
  • Level +: Words without a defined level yet

You can filter any page by level, and this selection stays consistent as you navigate.

Data sources

Most basic definitions and translations originate from Wiktionary. Wiktionary content is licensed under CC BY-SA 4.0, so this project may reuse and adapt it with attribution.

Notes

This project continues to evolve, and features such as improved flashcards, more examples, some grammar and smarter search are planned for the future.
